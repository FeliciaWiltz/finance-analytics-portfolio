Financial Performance Analysis

Analyzed company-wide financial performance by evaluating trends in budgeted vs actual spending over time. This project focuses on identifying performance patterns, growth trends, and key drivers of financial inefficiencies.

Objectives
Evaluate overall financial performance across multiple periods
Identify trends in spending and variance
Assess budget accuracy and financial efficiency
Highlight key contributors to financial performance

 Tools Used
Excel (Pivot Tables, Charts, Financial Metrics)

 Key Insights
Actual spending exceeded budget in multiple periods, indicating declining cost efficiency
Spending trends increased over time, suggesting potential seasonality or scaling operations
A small number of departments contributed disproportionately to total spending
Variability between budget and actual values highlights opportunities to improve forecasting accuracy
