
Budget vs Actual Financial Analysis

📌 Overview

Analyzed financial data to compare budgeted vs actual spending and identify key variance drivers across departments.

🎯 Objectives
Evaluate budget performance
Identify overspending trends
Provide actionable financial insights

🛠 Tools Used
Excel (Pivot Tables, Charts, Slicers)

📈 Key Insights
Marketing and Operations exceeded budget consistently
Spending increased over time, indicating potential seasonal trends
A small number of categories drove the majority of variance
