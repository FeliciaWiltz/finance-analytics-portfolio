Financial Forecasting Model

Developed a financial forecasting model using historical spending data to project future financial performance. The model includes scenario analysis to evaluate potential outcomes under different growth assumptions.

 Objectives
Forecast future financial performance based on historical trends
Calculate growth rates and identify patterns
Build scenario-based projections (best, base, worst case)
Support strategic planning through forward-looking analysis

Tools Used
Excel (Forecast Modeling and  Charts)

 Key Insights
Historical data shows a consistent growth trend in spending over time
Forecast projections indicate continued increases in financial activity under baseline assumptions
Scenario analysis highlights potential risks under lower growth conditions
Higher growth scenarios suggest increased financial exposure, requiring stronger cost controls
